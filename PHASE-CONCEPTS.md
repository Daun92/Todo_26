# PHASE-CONCEPTS.md - Mosaic Phase별 테마 & 컨셉 정의

> **"조각들이 모여 나만의 그림이 된다"**

---

## 🎯 전체 여정 개요

Mosaic의 4가지 Phase는 **성장의 순환 고리**를 형성합니다.

```
        ╭─────────────────────────────────────────╮
        │                                         │
        │     🔍 FEED ──────▶ 📚 LEARN           │
        │     (발견)           (내면화)           │
        │        ▲                 │              │
        │        │                 ▼              │
        │     📈 REFLECT ◀──── 🔗 CONNECT        │
        │     (성찰)           (연결)             │
        │                                         │
        ╰─────────────────────────────────────────╯

              끊임없이 순환하는 성장의 고리
```

각 Phase는 독립적이면서도 서로 연결되어,
**탐험 → 이해 → 연결 → 성찰**의 완전한 학습 사이클을 만듭니다.

---

## 🔍 Phase 1: FEED (발견)

### 테마
```
"호기심의 씨앗을 심다"
```

### 컨셉

**탐험가의 지도**

넓은 세상에서 흥미로운 것들을 발견하고 수집하는 단계입니다.
마치 탐험가가 미지의 땅에서 보물을 발견하듯,
다양한 분야의 콘텐츠를 만나고 "이건 배워보고 싶다"는
호기심의 불씨를 모읍니다.

### 핵심 철학

| 원칙 | 설명 |
|------|------|
| **넓게, 얕게** | 깊이보다 다양성. 많은 분야를 탐험 |
| **우연한 발견** | 계획된 학습보다 serendipity 중시 |
| **대척점 인식** | 내 관점과 반대되는 것도 함께 수집 |
| **부담 없이** | 모든 걸 다 배울 필요 없음. 일단 모으기 |

### 사용자 경험

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  😊 "오늘은 뭘 발견할 수 있을까?"                        │
│                                                         │
│  사용자는 마치 서점을 거닐듯                             │
│  다양한 콘텐츠를 훑어보고,                               │
│  끌리는 것들을 장바구니(학습 큐)에 담습니다.             │
│                                                         │
│  AI는 "이것도 재미있을 것 같아요" 하고                   │
│  의외의 것들을 슬쩍 추천합니다.                          │
│                                                         │
│  💡 "어? 이런 관점도 있네?" (대척점 발견)                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 목적

1. **학습 자료 발굴** - 다양한 소스에서 배울 거리 수집
2. **관심사 확장** - 기존 관심사 너머의 영역 탐험
3. **대척점 노출** - 편향을 깨는 반대 관점 접하기
4. **학습 동기 유발** - "이거 배우고 싶다" 욕구 생성

### 핵심 기능

| 기능 | 목적 | 상세 |
|------|------|------|
| **콘텐츠 수집** | 학습 자료 저장 | URL, 텍스트, 생각 메모 |
| **학습 큐** | 우선순위 관리 | 배우고 싶은 순서대로 정렬 |
| **AI 추천** | 세렌디피티 | 관심사 기반 + 의외의 발견 |
| **대척점 제안** | 균형 잡힌 시각 | 반대 관점 콘텐츠 함께 제시 |
| **태그 시스템** | 분류 & 연결 | 자유로운 태그로 분류 |

### AI 파트너 역할

```
🤖 "오늘 경제 기사를 많이 보셨네요.
    혹시 행동경제학에 대해 들어보셨어요?
    심리학과 경제학이 만나는 재미있는 분야예요."

🤖 "이 글과 반대 입장의 글도 찾아봤어요.
    두 관점을 비교해보면 어떨까요?"
```

### 성공 지표

- 수집한 콘텐츠 다양성 (분야 분포)
- 대척점 콘텐츠 비율
- 학습 큐 활용도
- 새로운 분야 탐험 빈도

## 📚 Phase 2: LEARN (내면화)

### 테마
```
"대화를 통해 내 것으로 만들다"
```

### 컨셉

**소크라테스식 대화**

수집한 콘텐츠를 진정으로 이해하고 내면화하는 단계입니다.
AI 파트너와의 인터뷰를 통해 단순히 읽는 것을 넘어,
질문에 답하고, 생각을 정리하며, 나만의 언어로 재구성합니다.

"읽었다"가 아닌 "이해했다"로 가는 여정.

### 핵심 철학

| 원칙 | 설명 |
|------|------|
| **대화로 배우기** | 혼자 읽기보다 대화가 깊은 이해를 만듦 |
| **질문의 힘** | 좋은 질문이 좋은 생각을 이끌어냄 |
| **나만의 언어** | 남의 말 인용이 아닌 내 언어로 표현 |
| **연결 짓기** | 새 지식을 기존 지식과 연결 |

### 사용자 경험

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  📖 콘텐츠를 선택하고 "학습 시작"을 누르면...           │
│                                                         │
│  🤖 AI: "이 글에서 가장 인상 깊은 부분이 뭐예요?"       │
│                                                         │
│  😊 나: "음... 여기서 말하는 복리 효과가..."            │
│                                                         │
│  🤖 AI: "좋아요! 그게 일상에서 어떻게 적용될 수 있을까요?"│
│                                                         │
│  😊 나: (생각하며) "아, 습관도 비슷하지 않을까?"         │
│                                                         │
│  🤖 AI: "오! 습관과 복리를 연결하셨네요. 더 말해줄래요?" │
│                                                         │
│  💡 대화하다 보니 스스로도 몰랐던 생각이 나온다!         │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 목적

1. **깊은 이해** - 표면적 읽기를 넘어 본질 파악
2. **능동적 학습** - 수동적 소비가 아닌 능동적 참여
3. **생각 정리** - 흩어진 생각을 구조화
4. **인사이트 발견** - 대화 중 새로운 깨달음

### 핵심 기능

| 기능 | 목적 | 상세 |
|------|------|------|
| **인터뷰 세션** | 대화형 학습 | AI 질문 → 사용자 답변 순환 |
| **음성 대화** | 자연스러운 표현 | Gemini Live로 말하며 학습 |
| **메모 작성** | 생각 기록 | 자유 형식 메모, 인사이트 저장 |
| **연결 질문** | 지식 통합 | "이전 학습과 어떻게 연결?" |
| **대척점 질문** | 균형 사고 | "반대 의견은 어떻게 생각?" |

### 인터뷰 질문 유형

```
1️⃣ 이해 확인
   "이 내용의 핵심을 한 문장으로 말한다면?"

2️⃣ 연결 질문
   "예전에 배운 것 중 이것과 연결되는 게 있나요?"

3️⃣ 대척점 질문
   "이 주장에 반대하는 사람은 뭐라고 할까요?"

4️⃣ 적용 질문
   "이걸 내 삶에 어떻게 적용할 수 있을까요?"

5️⃣ 감정 질문
   "이 내용을 읽으면서 어떤 느낌이 들었어요?"
```

### AI 파트너 역할

```
🤖 "잠깐, 방금 말한 거 정말 흥미로워요!
    '습관의 복리 효과'라고 표현하셨는데,
    이게 어떤 습관에서 느꼈어요?"

🤖 "좋은 통찰이에요. 근데 혹시 반대로,
    복리 효과가 부정적으로 작용하는 경우도 있을까요?"

🤖 "오늘 대화에서 발견한 것:
    '작은 습관도 복리처럼 쌓인다' - 메모해둘까요?"
```

### 성공 지표

- 인터뷰 완료율
- 세션당 평균 대화 깊이 (교환 수)
- 인사이트 생성 개수
- 학습 완료 후 메모 작성률

## 🔗 Phase 3: CONNECT (연결)

### 테마
```
"점들이 선이 되는 순간"
```

### 컨셉

**별자리 만들기**

개별적으로 보이던 지식들 사이에서 패턴과 연결고리를 발견하는 단계입니다.
마치 밤하늘의 별들을 이어 별자리를 만들듯,
서로 다른 분야의 학습들이 하나의 의미 있는 그림으로 연결됩니다.

이것이 바로 "Mosaic"의 핵심 - 조각들이 모여 나만의 그림이 됩니다.

### 핵심 철학

| 원칙 | 설명 |
|------|------|
| **크로스오버** | 다른 분야 간 연결이 가장 가치 있음 |
| **무의식 패턴** | 내가 모르는 나의 사고 패턴 발견 |
| **시각적 사고** | 그래프로 보면 보이지 않던 것이 보임 |
| **우연한 연결** | 의도치 않은 연결에서 통찰 발생 |

### 사용자 경험

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  🔗 지식 그래프를 바라보며...                            │
│                                                         │
│         [호흡법] ─── "집중력" ─── [투자]                │
│             │                       │                   │
│             └───── "스트레스" ──────┘                   │
│                        │                                │
│                    [명상]                               │
│                                                         │
│  😯 "어? 호흡법이랑 투자가 연결되어 있네?"              │
│                                                         │
│  🤖 AI: "흥미롭죠? 세 번이나 '집중력'으로 연결하셨어요. │
│         아마 판단을 내릴 때 집중력이 중요하다고         │
│         생각하시는 것 같아요."                          │
│                                                         │
│  💡 "내가 그렇게 생각하고 있었구나..."                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 목적

1. **패턴 발견** - 반복되는 연결 패턴 인식
2. **통합적 이해** - 분절된 지식을 하나로 통합
3. **무의식 인식** - 나도 몰랐던 사고방식 발견
4. **새 인사이트** - 연결에서 새로운 아이디어 도출

### 핵심 기능

| 기능 | 목적 | 상세 |
|------|------|------|
| **지식 그래프** | 시각화 | D3.js 인터랙티브 그래프 |
| **자동 연결 제안** | 연결 발견 | 태그/내용 기반 연결 제안 |
| **패턴 분석** | 무의식 발견 | 반복 연결 패턴 감지 |
| **크로스 인사이트** | 분야 간 연결 | 다른 분야 간 연결점 하이라이트 |
| **연결 스토리** | 의미 부여 | AI가 연결의 의미 해석 |

### 연결 유형

```
═══════════════════════════════════════════════════════

📎 관련 (Related)
   "비슷한 주제를 다룸"
   예: [호흡법] ─ 관련 ─ [명상]

🔄 원인-결과 (Cause-Effect)
   "A가 B에 영향을 줌"
   예: [수면 부족] ─ 원인 ─ [판단력 저하]

⚖️ 대척점 (Counterpoint)
   "서로 반대되는 관점"
   예: [성장 투자] ─ 대척점 ─ [가치 투자]

🌉 크로스오버 (Crossover)
   "다른 분야 간 연결"
   예: [심리학] ─ 크로스 ─ [경제학]

💡 인사이트 (Insight)
   "새로운 깨달음으로 연결"
   예: [습관] ─ 인사이트 ─ [복리 효과]

═══════════════════════════════════════════════════════
```

### AI 파트너 역할

```
🤖 "재미있는 패턴을 발견했어요!
    경제 관련 글을 읽을 때마다 심리학으로 연결하시네요.
    혹시 행동경제학에 관심 있으세요?"

🤖 "이 두 개념은 완전 다른 분야인데,
    '결정 피로'라는 공통점이 있어요.
    연결해볼까요?"

🤖 "당신의 지식 그래프가 점점 풍성해지고 있어요.
    특히 '마음 챙김' 클러스터가 인상적이에요."
```

### 성공 지표

- 생성된 연결 수
- 크로스오버 연결 비율
- 패턴 발견 횟수
- 그래프 탐색 시간

## 📈 Phase 4: REFLECT (성찰)

### 테마
```
"거울을 보며 성장을 확인하다"
```

### 컨셉

**성장의 일기**

일정 기간 동안의 학습 여정을 돌아보고,
얼마나 성장했는지, 어떤 패턴이 있는지,
어디로 더 나아갈 수 있는지 성찰하는 단계입니다.

숲에서 잠시 나와 전체 그림을 바라보는 시간.
AI 파트너가 당신만의 성장 스토리를 함께 써줍니다.

### 핵심 철학

| 원칙 | 설명 |
|------|------|
| **메타인지** | 배운 것뿐 아니라 '어떻게 배우는지' 인식 |
| **균형 점검** | 편향된 학습 영역 인식 및 조정 |
| **작은 성공** | 큰 목표보다 작은 성취 축하 |
| **방향 재설정** | 회고를 통한 다음 방향 조정 |

### 사용자 경험

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  📊 12월 성장 리포트                                     │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  📚 15개 콘텐츠  💬 8회 인터뷰  ✏️ 42개 메모    │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  🔍 발견한 연결고리                                      │
│  • '판단력'이 경제, 심리, 건강 모든 분야에 등장          │
│  • 아침 시간대에 가장 깊은 학습을 함                     │
│                                                         │
│  ⚖️ 편향 분석                                           │
│  • 경제 분야 학습 70% (편중 주의)                       │
│  • 추천: 철학, 예술 분야 탐험해보기                      │
│                                                         │
│  📖 AI의 성장 스토리:                                    │
│  "이번 달 당신은 '판단력'이라는 키워드를 중심으로       │
│   다양한 분야를 탐험했습니다. 특히 경제적 결정에서      │
│   심리적 요소가 중요하다는 것을 깨달은 것 같아요.       │
│   다음 달에는 이 통찰을 실제 생활에 적용해보면          │
│   어떨까요?"                                            │
│                                                         │
│  😊 "내가 이렇게 성장하고 있었구나..."                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 목적

1. **성장 확인** - 객관적 데이터로 성장 시각화
2. **편향 인식** - 나의 알고리즘과 편향 점검
3. **방향 조정** - 다음 학습 방향 설정
4. **동기 부여** - 작은 성공 축하, 지속 동력

### 핵심 기능

| 기능 | 목적 | 상세 |
|------|------|------|
| **학습 통계** | 양적 파악 | 콘텐츠, 세션, 메모, 연결 수 |
| **성장 타임라인** | 시간 흐름 | 일별/주별/월별 학습량 시각화 |
| **편향 분석** | 균형 체크 | 분야별 분포, 편중 경고 |
